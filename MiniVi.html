A MiniVi Line Editor...

My first of many programming projects to come. I and two other colleagues worked together using many files & data structures including: arrays, stacks, linked lists, linked stacks, etc. in C++ to create a simple-to-use line editor.

First Iteration...

When just beginning to work on this project we faced four tasks:

    • 1.) Read lines from files
      2.) Delete characters from a line
      3.) User can save file or quit editor without saving
      4.) Places cursor at (0,0)

    • Second Iteration...

    Once our first iteration passed all four previous tasks, we were then cleared to proceed to the next tasks:

    • 1.) User can move cursor up, left, down, and right ('w', 'a', 's', 'd')
      2.) Delete lines from editor
      3.) Asks the user if they are sure they want to quit after changes are made to editor

    • Third Iteration...

    Once our second iteration passed all three previous tasks, we were then cleared to proceed to the next tasks:

    • 1.) Undo changes
      2.) Color Text
      3.) Implement keyword text file
      4.) Use Binary Search to sort keywords

    • Fourth Iteration...

    Once our third iteration passed all four previous tasks, we were then cleared to proceed to the next task:

    • 1.) Use Binary Search Tree to sort keywords
